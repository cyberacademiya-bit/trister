## Авто 24/7
GitHub Actions (`.github/workflows/bot.yml`) — два расписания:
- **Каждый час** (`0 * * * *`): живая торговля + **проверка стоп-лосса** («продать при риске»). Фьючерсный бот пишет каждый стоп/сделку в журнал `trades.csv`.
- **Раз в день** (`0 14 * * *` = 20:00 Бишкек): дополнительно тяжёлый турнир всех стратегий + локальный симулятор.

Мак может быть выключен. Для публичного репо минуты Actions **бесплатны** → почасовой режим ничего не стоит.
Вход в позицию — по дневным свечам (меньше дёрганья), а стоп-лосс проверяется каждый час по живой цене.
