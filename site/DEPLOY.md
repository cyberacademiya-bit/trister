# Деплой дашборда на Vercel

Дашборд — статичный HTML (снимок данных), деплоится за минуту.

## Способ 1 — через сайт (проще всего)
1. Зайди на https://vercel.com → **Add New → Project** (или **Deploy**)
2. Перетащи папку `site/` (или залей через drag-and-drop на vercel.com/new)
3. Готово — Vercel даст ссылку вида `трister-xxx.vercel.app`

## Способ 2 — через CLI
```bash
npm i -g vercel      # один раз
cd site
vercel               # войдёшь в аккаунт и задеплоит
vercel --prod        # на боевой домен
```

## Как обновить данные (снимок → свежий)
Данные «зашиты» в страницу на момент сборки. Чтобы обновить:
```bash
cd ..
python gather_data.py          # собрать свежие балансы/сделки
# затем пересобрать dashboard.html (Клод сделает) и снова:
cp dashboard.html site/index.html
vercel --prod                  # передеплой
```
